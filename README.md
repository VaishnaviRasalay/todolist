# Todo List App (React)

A simple Todo List application built using React.

## Features
- Add, edit, delete todos
- Mark todos as completed
- Search functionality
- Component-based architecture
- Clean and colorful UI

## Tech Stack
- React
- JavaScript
- CSS

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
A React Todo List application implementing CRUD operations, search, and task completion using hooks and component-based architecture.
